О проекте#

Ментор МГПУ - это проект по созданию цифрового наставника на основе искусственного интеллекта и веб-приложения для молодых преподавателей, предоставляющего легкий доступ к консультациям и индивидуальной поддержке.

О продукте#

  1. Формат реализации

    • Веб-сайт с возможностью преобразования в PWA (Progressive Web Application).
    • Кросс-платформенная совместимость (работает на любых устройствах).

  2. Технологический стек

    • Frontend: Веб-интерфейс чата (аналог ChatGPT/Deepseek, но с упрощённым дизайном).

    • Backend: Python + LangChain (работа с LLM и RAG). ChromaDB (векторное хранилище для семантического поиска). SQLite (хранение пользовательских данных: аутентификация, история чатов).

    • Аутентификация: VK OAuth 2.0 (упрощённый вход без собственной системы регистрации).

  3. Ключевые функции:

    • ✅ Чат с LLM (базовый интерфейс для взаимодействия с моделью).
    • ✅ Авторизация через VK (безопасность + экономия времени на разработку).
    • ✅ Сохранение истории чатов (но пока без контекстного использования в диалогах).
    • ✅ Retrieval-Augmented Generation (RAG).
    • Загрузка и обработка пользовательских документов.
    • Векторизация текста (ChromaDB).
    • Семантический поиск перед генерацией ответа.

  4. Особенности MVP (Minimal Viable Product (минимально жизнеспособный продукт))

    • Упрощённый дизайн (фокус на функциональность, а не на визуал)
    • Один чат на пользователя (без сложной системы тредов).
    • Контекст не учитывается (ответы генерируются на основе одного запроса).
